# 🐄 CowSay Kids

**A fun ASCII cow that solves math expressions and shares amazing facts** — pick a category and learn!

## Quick Start

Open `index.html` in any browser — no install, no server.

## Features

- ASCII art cow with speech bubbles
- Math expression solver
- Fun facts by category
- Trilingual support (EN/FR/AR)
- Single-file HTML app, zero dependencies
